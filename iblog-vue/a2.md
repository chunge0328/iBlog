## Vue
Vue和Angular差不多，是一个MVVM框架，并且它俩的语法规则很相似，以前学过Angular，感觉学习Vue很轻松。

## Vue的基本用法
在html中先引入

```
<script src="vue.js"></script>
    <script src="vue-resource.js"></script>
</code>
```
    
然后再js中

  
```
var c=new Vue({
		el:'body',	//选择器  class tagName
		data:{
		    //这里生命变量
		    msg:'welcome vue'
		},
		methods:{
		    //这里声明方法
		}
	});
```


这样，就可以使用Vue了

## Vue的常用指令
- v-model	一般表单元素(input)	双向数据绑定
- 循环
    - v-for="name in arr"
    - v-for="name in json"
- 事件
    - v-on:click="函数"
    - 简写成：@click="函数"

- 可以阻止冒泡
    - ev.cancelBubble=true;
	- @click.stop	推荐
- 可以阻止默认事件
    - ev.preventDefault();
    - @contextmenu.prevent	推荐

## 绑定变量

    
```
v-bind:src="变量名"
		width/height/title....
	
	简写:
	:src="变量名"	推荐

	<img src="{{url}}" alt="">	效果能出来，但是会报一个404错误
	<img v-bind:src="url" alt="">	效果可以出来，不会发404请求
```

	
## 模板2
- {{msg}}		数据更新模板变化
- {{*msg}}	数据只绑定一次
- {{{msg}}}	HTML转意输出

## 样式
- :class=""	<=> v-bind:class=""
- :style=""	<=> v-bind:style=""
- :class="[red]" => red是数据
- :class="[red,b,c,d]"
- :class="{red:a, blue:false}"
- :class="json" =>
		data:{
			json:{red:a, blue:false}
		}
- :style="[c]"
- :style="[c,d]"
注意:  复合样式，采用驼峰命名法
- :style="json"
